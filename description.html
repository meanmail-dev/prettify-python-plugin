Prettify Python

Enhance the readability of your Python code with beautiful mathematical symbols!

This plugin automatically replaces standard Python operators with their mathematical equivalents, making your code more elegant and readable.

Key Features

  • Automatic operator replacement with mathematical symbols
  • Customizable replacement rules
  • Instant changes application

How to Use

  1. Install the plugin
  2. Open any Python file
  3. Enjoy automatic formatting!

Configuration

You can customize replacement rules in the plugin settings:

  1. Go to Settings/Preferences
  2. Find "Prettify Python" in the plugins list
  3. Here you can:
    • Enable/disable specific replacements
    • Add new custom replacements
    • Edit existing replacements
    • Organize replacements by categories
    • Import/Export your replacement settings

Note: If you're using a font with ligatures, you can disable the replacements that your font already provides and keep only the ones you need!
